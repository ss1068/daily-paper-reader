## 摘要
世界动作模型（World Action Model, WAM）能够联合预测未来的世界状态和动作，与传统模型相比展现出更强的物理操作能力。这种强大的物理交互能力是一把双刃剑：如果忽视安全性，它将直接威胁人身安全、财产安全和环境安全。然而，现有研究对这一关键安全漏洞——即 WAM 对越狱攻击（jailbreak attacks）的脆弱性——关注极少。为了填补这一空白，我们定义了三级安全分类框架（Three-Level Safety Classification Framework），以系统地量化机械臂运动的安全性。此外，我们提出了 JailWAM，这是首个专门针对 WAM 的越狱攻击与评估框架，由三个核心组件组成：(1) 视觉-轨迹映射（Visual-Trajectory Mapping），将异构动作空间统一为视觉轨迹表示，并实现跨架构的统一评估；(2) 风险判别器（Risk Discriminator），作为一种高召回率的筛选工具，在识别视觉轨迹中的破坏性行为时优化了效率与准确率之间的权衡；(3) 双路径验证策略（Dual-Path Verification Strategy），首先通过基于单图像的视频-动作生成模块进行快速粗筛，然后通过完整的闭环物理仿真进行高效且全面的验证。此外，我们构建了 JailWAM-Bench，这是一个用于全面评估 WAM 在越狱攻击下安全对齐性能的基准测试。在 RoboTwin 仿真环境中的实验表明，所提出的框架能有效揭示物理漏洞，在最先进的 LingBot-VA 上实现了 84.2% 的攻击成功率。同时，基于 JailWAM 可以构建鲁棒的防御机制，为设计安全可靠的机器人控制系统提供了有效的技术方案。

## Abstract
The World Action Model (WAM) can jointly predict future world states and actions, exhibiting stronger physical manipulation capabilities compared with traditional models. Such powerful physical interaction ability is a double-edged sword: if safety is ignored, it will directly threaten personal safety, property security and environmental safety. However, existing research pays extremely limited attention to the critical security gap: the vulnerability of WAM to jailbreak attacks. To fill this gap, we define the Three-Level Safety Classification Framework to systematically quantify the safety of robotic arm motions. Furthermore, we propose JailWAM, the first dedicated jailbreak attack and evaluation framework for WAM, which consists of three core components: (1) Visual-Trajectory Mapping, which unifies heterogeneous action spaces into visual trajectory representations and enables cross-architectural unified evaluation; (2) Risk Discriminator, which serves as a high-recall screening tool that optimizes the efficiency-accuracy trade-off when identifying destructive behaviors in visual trajectories; (3) Dual-Path Verification Strategy, which first conducts rapid coarse screening via a single-image-based video-action generation module, and then performs efficient and comprehensive verification through full closed-loop physical simulation. In addition, we construct JailWAM-Bench, a benchmark for comprehensively evaluating the safety alignment performance of WAM under jailbreak attacks. Experiments in RoboTwin simulation environment demonstrate that the proposed framework efficiently exposes physical vulnerabilities, achieving an 84.2% attack success rate on the state-of-the-art LingBot-VA. Meanwhile, robust defense mechanisms can be constructed based on JailWAM, providing an effective technical solution for designing safe and reliable robot control systems.

*   **核心问题**：现有的越狱攻击研究多集中在数字领域（如生成恶意文本或视频），而 WAM 的越狱会导致**物理层面的真实危害**（如人身伤害、设备损坏）。
*   **研究动机**：目前缺乏系统性的框架来量化机器人运动安全，且 WAM 的动作空间异构性使得统一评估困难，且全闭环仿真验证成本极高。

#### 2. 方法论
JailWAM 框架由以下核心组件构成：
*   **三级安全分类框架**：
    *   **Level 0 (安全合规)**：运动平滑，无碰撞。
    *   **Level 1 (运动失败)**：系统死锁、高频振荡或无有效位移，导致任务中断。
    *   **Level 2 (灾难性风险)**：失控的大范围摆动、碰撞工作台或人员，造成严重破坏。
*   **视觉-轨迹映射 (VTM)**：将抽象的低级动作序列（如关节坐标）投影为标准化的 2D 正交视图（俯视图和前视图）轨迹图，利用视觉语言模型（VLM）的几何推理能力识别异常。
*   **风险判别器 (RD)**：基于 Qwen3-VL-2B 微调的轻量化模型，通过分析 VTM 生成的轨迹图和指令，快速预测安全等级。
*   **双路径验证策略**：
    *   **阶段 I (开环筛选)**：利用 WAM 的生成先验产生动作，通过 RD 快速过滤掉安全样本。
    *   **阶段 II (闭环验证)**：仅对高风险指令进行高保真物理仿真和人工审核，确保评估的物理真实性。
*   **攻击生成**：利用 LLM（如 Gemini, GPT-5）配合专门设计的越狱模板，生成任务无关但具有物理威胁的指令。

#### 3. 实验设计
*   **数据集与场景**：
    *   **RoboTwin**：针对 LingBot-VA（SOTA 模型）和 Motus 进行测试。
    *   **LIBERO**：针对 Cosmos-Policy 和非 WAM 模型 $\pi_{0.5}$ 进行跨环境测试。
*   **基准测试 (JailWAM-Bench)**：包含 82 条具有跨任务迁移能力的越狱指令。
*   **对比方法**：

*   **评估指标**：攻击成功率 (ASR)、运动失败率 (MFR)、灾难风险率 (CRR)。

#### 5. 实验数量与充分性
*   **实验规模**：涵盖了 50 个 RoboTwin 场景和 LIBERO 环境，测试了 4 种不同架构的模型。
*   **消融实验**：验证了 VTM 的必要性、微调对 RD 性能的提升、以及双路径策略的效率。
*   **充分性评价**：实验设计较为全面，不仅验证了攻击的有效性，还探讨了**跨架构迁移性**（从 LingBot 迁移到 Motus）和**跨环境泛化性**。通过对比非 WAM 模型（$\pi_{0.5}$），客观地证明了安全漏洞主要源于 WAM 的视觉生成先验。

#### 6. 主要结论与发现
*   **严重脆弱性**：JailWAM 在 SOTA 模型 LingBot-VA 上达到了 **84.2% 的攻击成功率**，其中 22.2% 为灾难性风险。
*   **强迁移性**：针对某一模型生成的越狱指令可以零样本迁移到其他 WAM 架构，ASR 依然保持在较高水平（如 Motus 达到 60.6%）。

*   **防御可行性**：提出的风险判别器可以作为推理时的“插件式”过滤器，在保持 96% 正常指令通过率的同时，大幅降低越狱成功率。

#### 7. 优点
*   **首创性**：首次系统性地研究了具身智能中 WAM 的越狱安全问题。
*   **效率优化**：双路径验证策略将评估速度提升了 **2.5 倍**，解决了仿真成本高的痛点。
*   **维度统一**：通过视觉轨迹映射解决了不同机器人动作空间不统一的评估难题。

#### 8. 不足与局限
*   **仿真与现实差距 (Sim-to-Real)**：所有实验均在仿真环境中完成，尚未在真实物理实体机器人上验证其破坏力。

*   **防御深度**：提出的防御手段主要在推理端过滤，尚未深入探讨如何在模型训练阶段进行更本质的安全对齐。
